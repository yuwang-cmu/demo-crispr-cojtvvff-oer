# Chapter 2 Practice: Editing Genomes

:::warning
**Demonstration only:** This practice set shows what Coursewerk can produce. It is not intended for direct use in actual teaching.
:::

These 15 items cover every chapter objective. Attempt each question before revealing the worked answer.

**(Objective 1 · workflow reconstruction)**

:::: tabs
::: tab Q
Put these stages in a defensible order: measure molecular outcomes; define the claim; choose an editor and delivery form; test biological function; design controls.
:::
::: tab Answer
Define the claim → choose an editor and delivery form → design controls → measure molecular outcomes → test biological function. In practice, the workflow is iterative, but the claim must precede interpretation and molecular change must be distinguished from functional consequence.
:::
::::

**(Objective 1 · missing-stage critique)**

:::: tabs
::: tab Q
A report says, “Sequencing shows the intended nucleotide at the target, so the disease phenotype has been corrected.” Identify the missing inferential steps.
:::
::: tab Answer
Target-site sequencing does not by itself establish correct transcript or protein behavior, restoration of cellular function, organism-level benefit, durability, delivery safety, or acceptable unintended changes. Those require additional assays and controls at progressively higher levels of the evidence ladder.
:::
::::

**(Objective 2 · allele distribution)**

:::: tabs
::: tab Q
Among 100 sampled molecules, 55 contain the intended allele, 25 contain small insertions or deletions, 10 remain unchanged, and 10 fall into other sequence classes. What is the intended-edit fraction, and why is “55% successful cells” not justified?
:::
::: tab Answer
The intended allele represents 55% of the sampled molecules. Molecules are not necessarily equivalent to cells: ploidy, multiple alleles, sampling, amplification, and mosaic combinations can separate molecule fractions from the fraction of cells with a desired genotype.
:::
::::

**(Objective 2 · NHEJ interpretation)**

:::: tabs
::: tab Q
Repair this statement: “Non-homologous end joining always creates an error.”
:::
::: tab Answer
“End joining can restore the original junction or produce insertions, deletions, or other junction outcomes; the distribution depends on lesion and cellular context.” Calling the entire pathway uniformly error-prone hides both precise rejoining and reproducible biases.
:::
::::

**(Objective 2 · pathway prediction)**

:::: tabs
::: tab Q
Repeated deletions share endpoints near short matching sequences. Which repair model deserves investigation, and does the pattern prove it?
:::
::: tab Answer
Microhomology-mediated end joining deserves investigation because short matching sequences can bias deletion endpoints. The pattern is suggestive, not conclusive; competing mechanisms and measurement artifacts must be tested.
:::
::::

**(Objective 2 · conditional HDR)**

:::: tabs
::: tab Q
Why can providing a donor template fail to make homology-directed repair the dominant outcome?
:::
::: tab Answer
Template availability is only one condition. Cell state and cell-cycle phase, delivery timing, lesion type, competing repair pathways, template design, toxicity, and locus context can all affect whether template-directed repair occurs.
:::
::::

**(Objective 2 · newer editors)**

:::: tabs
::: tab Q
Base and prime editors can avoid a programmed double-strand break in many designs. Why does that not make their outcomes automatically precise or consequence-free?
:::
::: tab Answer
They can still produce mixed product distributions, bystander edits, incomplete conversion, unintended DNA or RNA changes, delivery effects, and context-dependent biological consequences. Avoiding one lesion class changes the risk profile; it does not eliminate the need for validation.
:::
::::

**(Objective 3 · delivery trade-offs)**

:::: tabs
::: tab Q
Compare plasmid DNA, editor mRNA, and preassembled ribonucleoprotein (RNP) delivery along the dimension of exposure duration.
:::
::: tab Answer
Plasmid DNA can support more prolonged expression, mRNA is transient but must be translated, and an RNP supplies active components directly and is often more transient. Actual duration depends on cell type, dose, carrier, and experimental context, so this is a comparative tendency rather than a universal ranking.
:::
::::

**(Objective 3 · confounding diagnosis)**

:::: tabs
::: tab Q
Edited cells grow more slowly than untreated cells. What control helps distinguish editor-dependent effects from delivery stress?
:::
::: tab Answer
Use a mock-delivery control that receives the same carrier and procedure without the active editing payload. Additional useful controls include a non-targeting guide and an editor-only condition, chosen to isolate different components of the intervention.
:::
::::

**(Objective 4 · assay boundary)**

:::: tabs
::: tab Q
What can deep sequencing of a short target-site amplicon measure well, and what important dimensions can it miss?
:::
::: tab Answer
It can estimate sequence classes within the amplified interval among molecules that amplify successfully. It can miss large rearrangements, distant changes, amplification dropout, delivery toxicity, cell-to-cell combinations, transcript or protein effects, and biological function.
:::
::::

**(Objective 4 · quality dashboard)**

:::: tabs
::: tab Q
Classify each metric: (a) fraction with intended target allele; (b) diversity of unintended target-site products; (c) changes at other genomic sites; (d) restored protein activity.
:::
::: tab Answer
(a) on-target efficiency; (b) on-target heterogeneity; (c) off-target activity; (d) functional outcome. None of these metrics alone substitutes for the others.
:::
::::

**(Objective 4 · interpretation repair)**

:::: tabs
::: tab Q
A treatment has 90% target-site modification and poor cell survival. Explain why “highly effective” is ambiguous.
:::
::: tab Answer
It may be efficient at creating a molecular change among measured molecules, yet ineffective or unacceptable at preserving viable cells or producing the intended function. “Effective” must name the outcome level, denominator, and safety context.
:::
::::

**(Objective 5 · evidence ladder)**

:::: tabs
::: tab Q
Order these claims from lower to higher biological scope: altered target DNA; altered protein; altered transcript; changed cell function.
:::
::: tab Answer
Altered target DNA → altered transcript → altered protein → changed cell function. The order is an evidence ladder, not a guarantee: each transition needs its own measurements and controls.
:::
::::

**(Objective 5 · control selection)**

:::: tabs
::: tab Q
Two different guides produce the same phenotype, and restoring an uneditable version of the gene reverses it. How does this strengthen causal interpretation?
:::
::: tab Answer
Independent guides reduce the chance that one guide-specific unintended event explains the phenotype. Rescue with an uneditable gene version tests whether loss or alteration of the intended gene is causally connected to the phenotype. Neither control proves perfect specificity, but together they strengthen the chain.
:::
::::

**(Objective 6 · empirical and normative reasoning)**

:::: tabs
::: tab Q
Sort these as primarily empirical or normative: (a) measured mosaicism rate; (b) whether heritable editing should be permitted; (c) off-target variants detected by an assay; (d) what level of uncertainty society should accept.
:::
::: tab Answer
(a) empirical; (b) normative; (c) empirical within the assay’s limits; (d) normative, informed by empirical evidence. Responsible governance requires both kinds of reasoning without pretending that measurements alone settle value judgments.
:::
