# Chapter 1 Assessment Guide: CRISPR as a Biological System

> **Demonstration only:** This AI-generated Coursewerk guide is not intended for direct use in actual teaching.

## Source and format

Adapted and reorganized from the revision-bound Wikipedia sources in `metadata/SOURCE_RECORD.json` under CC BY-SA 4.0. This is assessment-design guidance, not a secure question bank or laboratory protocol.

## General assessment priorities

- Reward precise use of **array**, **repeat**, **spacer**, **protospacer**, **PAM**, **guide**, and **effector**.
- Use causal diagrams, sequence annotations, claim critique, and transfer cases rather than recall alone.
- Require qualifiers when a statement applies specifically to SpCas9 or Type II systems.
- Treat predictions about activity as hypotheses to test, not guarantees from sequence alone.

## Objective 1: Distinguish CRISPR arrays, Cas proteins, guide RNAs, and “CRISPR–Cas9”

### Target understanding

Learners identify the genomic array as an information record, Cas9 as a protein effector, and guide RNA as a targeting component without using the terms interchangeably.

### Question guides

#### Guide A — Component sorting
- **Variables:** 6–10 descriptions; include structure, location, and function.
- **Constraint:** every answer needs a one-clause justification.
- **Worked instantiation:** “Contains alternating repeated and variable sequences” → CRISPR array, not Cas9.

#### Guide B — Repair the metaphor
- **Variables:** headline or diagram using “CRISPR scissors.”
- **Constraint:** preserve readability while restoring guide, PAM, effector, and cellular-response roles.
- **Worked instantiation:** revise “CRISPR finds a bad gene” to “a guide-bearing Cas effector can recognize a compatible target context selected by the designer.”

|---|---|---|
| Homework | Analyze | Annotate a popular explanation. |
| Quiz | Understand | Sort components with rationales. |
| Exam | Evaluate | Repair a misleading model. |
| Project | Create | Build a layered glossary diagram. |

## Objective 2: Trace adaptation, guide biogenesis, and interference

### Target understanding

Learners order foreign-sequence capture, array integration, transcription, processing, effector loading, target recognition, and interference while recognizing that system details vary.

### Question guides

#### Guide A — Reconstruct the pathway
- **Variables:** 6–9 shuffled stages; omit one stage in advanced versions.
- **Constraint:** arrows must use causal verbs.
- **Worked instantiation:** protospacer selection → spacer acquisition → array transcription → crRNA processing → effector loading → interference.

#### Guide B — Stage failure
- **Variables:** one component disabled at acquisition, processing, or interference.
- **Constraint:** distinguish effect on new memory from effect on use of existing spacers.
- **Worked instantiation:** loss of Cas1–Cas2 can block new acquisition without necessarily erasing already stored spacers.

| Purpose | Cognitive level | Design |
|---|---|---|
| Homework | Apply | Predict three stage-specific disruptions. |
| Discussion | Analyze | Compare “memory” and “response.” |
| Quiz | Understand | Order stages. |
| Exam | Analyze | Diagnose a missing function from data. |

## Objective 3: Relate spacer, protospacer, and PAM

### Target understanding

Learners locate the spacer in the host array, the protospacer in foreign or target nucleic acid, and the PAM beside the protospacer for PAM-dependent systems.

### Question guides

#### Guide A — Sequence annotation
- **Variables:** arrays of 3–6 repeat–spacer units and 2–4 candidate targets.
- **Constraint:** label strand orientation and keep PAM outside the guide-pairing region.
- **Worked instantiation:** a host spacer matches a phage protospacer; the adjacent motif is annotated separately.

#### Guide B — Claim boundary
- **Variables:** candidate with match/no PAM, candidate with PAM/mismatch.
- **Constraint:** no answer may use “guaranteed.”
- **Worked instantiation:** perfect complementarity without compatible PAM predicts weak ordinary SpCas9 targeting, but is not a universal CRISPR conclusion.

|---|---|---|
| Homework | Apply | Annotate host and invader sequences. |
| Quiz | Understand | Locate the PAM. |
| Exam | Analyze | Compare target candidates. |
| Project | Create | Write an accessible target legend. |

## Objective 4: Model Cas9 targeting as checkpoints

### Target understanding

Learners explain PAM sampling, local DNA opening, guide pairing, conformational activation, and cleavage as separable steps affected by context.

### Question guides

#### Guide A — Checkpoint evidence
- **Variables:** binding, cleavage, or expression data for 3–5 targets.
- **Constraint:** infer only the stage directly supported.
- **Worked instantiation:** binding without cleavage supports recognition but not nuclease activation.

#### Guide B — Mismatch reasoning
- **Variables:** mismatch position, number, chromatin context, editor exposure.
- **Constraint:** require a predicted trend plus a proposed measurement.
- **Worked instantiation:** a mismatch may reduce activity, but its effect is an empirical question for that guide–target pair.

| Purpose | Cognitive level | Design |
|---|---|---|
| Homework | Analyze | Map observations to checkpoints. |
| Discussion | Evaluate | Compare two specificity claims. |
| Quiz | Apply | Choose the supported checkpoint. |
| Exam | Evaluate | Design a discrimination experiment. |

## Objective 5: Compare active Cas9, nickase, and dCas9

### Target understanding

Learners separate guide-directed recognition from double-strand cleavage, single-strand nicking, and binding-based regulation.

### Question guides

#### Guide A — Platform selection
- **Variables:** desired action: break, paired nick, repression, activation, or localization.
- **Constraint:** explain which molecular function is retained and changed.
- **Worked instantiation:** promoter repression without sequence change suggests dCas9-based binding rather than active nuclease use.

#### Guide B — Data interpretation
- **Variables:** sequence, transcript, and protein measurements.
- **Constraint:** match the observed layer to the platform claim.
- **Worked instantiation:** unchanged DNA plus reduced transcript is compatible with CRISPR interference but not proof of a knockout.

| Purpose | Cognitive level | Design |
|---|---|---|
| Homework | Apply | Match platforms to intended action. |
| Quiz | Understand | Identify retained functions. |
| Exam | Analyze | Reconcile sequence and expression data. |
| Project | Create | Build a modular platform matrix. |

## Objective 6: Predict effects of guide, PAM, and target-context changes

### Target understanding

Learners form conditional predictions and identify measurements needed to test them, avoiding universal rankings across guides or cell types.

### Question guides

#### Guide A — Controlled comparison
- **Variables:** change one of guide sequence, PAM, mismatch, or accessibility while holding others fixed.
- **Constraint:** identify the controlled and changed variables.
- **Worked instantiation:** changing only the guide segment is intended to redirect recognition while retaining the Cas9-binding scaffold.

#### Guide B — Overclaim audit
- **Variables:** 20–40-word specificity claim.
- **Constraint:** rewrite with scope, assay, and uncertainty.
- **Worked instantiation:** replace “no off-targets” with “none were detected among the assayed candidate sites at the stated detection limit.”

| Purpose | Cognitive level | Design |
|---|---|---|
| Homework | Analyze | Predict and test one variable at a time. |
| Discussion | Evaluate | Audit specificity language. |
| Quiz | Apply | Identify controlled variables. |
| Exam | Create | Propose a bounded comparison. |

## Rubric themes

1. Component and sequence vocabulary is accurate.
2. Causal stages are complete and correctly ordered.
3. Type II/SpCas9 examples are not generalized to all CRISPR systems.
4. Claims distinguish prediction, observation, and inference.
5. Uncertainty and assay limits are explicit.
