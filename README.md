# CRISPR and Genome Editing

> **Demonstration only:** this package shows what an AI agent can produce by following Coursewerk. It is not intended for direct use in an actual class and has not received subject-matter or classroom validation.

This two-chapter package moves from CRISPR's natural role in microbial immunity to the design, outcome, and evaluation of genome-editing experiments. The organizing idea is that every editing claim is a chain: a biological recognition system is engineered into a reagent, the reagent creates a molecular event, cellular repair generates a population of outcomes, and evidence determines which conclusions are justified.

Each chapter contains a concept map, study guide, slide deck, assessment-support guide, and 15 worked practice problems. Original accessible SVG diagrams are complemented by independently licensed documentary images. The default built-in themes are `light-academic-2` for reading pages and `sage` for slides.
